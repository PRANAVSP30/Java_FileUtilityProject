### Project Overview

This project demonstrates basic file operations in Java using:

  FileWriter

  FileReader

  BufferedReader

  PrintWriter

  Scanner

The program allows the user to:

    Create / Write to a file
    Read an existing file
    Modify & append content
    Exit safely

All operations are performed through a menu-driven console application.


### Menu Options

    1️. Write to File
    Creates a new file or overwrites the existing file.

    2️. Read File
    Displays the content of the file.

    3️. Modify File
    Appends new content to the existing file.

    4️. Exit
    Terminates the application gracefully.

### Tech Stack
- Java (Core Java)
- File Handling (FileWriter, FileReader, BufferedReader, PrintWriter)
- Exception Handling & I/O Streams
- Menu-driven console application
- Tools: VS Code, Java JDK 25, Git & GitHub
